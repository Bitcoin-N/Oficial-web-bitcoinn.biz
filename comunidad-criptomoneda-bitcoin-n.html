Desarrollo

Bitcoin-N es un proyecto complejo y en constante evolución, por lo que nos entusiasma el nuevo código y damos la bienvenida a todos los desarrolladores con experiencia a participar y contribuir en el código. Tenemos muchos planes futuros para mejoras y optimizaciones de Bitcoin-N que no podremos hacer sin un sólido equipo de desarrollo de código abierto.
El repositorio del proyecto está alojado en github si quieres verificarlo o contribuir.

Documentación

El wiki del proyecto es un wiki mantenido por y para la comunidad, con mucha información técnica sobre como funciona Bitcoin-N.

Puedes ayudarnos a mejorarlo corrigiendo errores, agregando traducciones y manteniéndolo actualizado.

Notificar errores

Puedes usar el rastreador de errores en github si quieres informar de cualquier error y ayudarnos a hacer que Bitcoin-N sea el más estable.
